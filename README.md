## JoybaitGPT

It absolutely adores talking to you and tries to spread joy with every response!

Built with React, Express and Ollama with gemma3

A small project to refine my react skills while exploring the domain of AI engineering

## Instructions

Install Ollama: https://ollama.com/
Download gemma3

```
git clone git@github.com:Goooble/ollama-chatbot.git
npm init
node index.js
npm run dev
```
